# 🎓 Calculadora de Álgebra Universitaria

Una herramienta web interactiva diseñada para resolver operaciones complejas de álgebra lineal y geometría analítica, mostrando el desarrollo matemático paso a paso. Ideal para estudiantes universitarios de ingeniería y ciencias exactas.

## 🚀 Características Principales

* **Matrices:** Cálculo de Producto (A × B), Matriz Traspuesta (Aᵀ), Matriz Inversa (A⁻¹) y Determinantes (|A|).
* **Vectores (R³):** Cálculo de Producto Escalar (u · v), Producto Vectorial (u × v), Ángulo entre vectores (θ) y Proyección ortogonal.
* **Desarrollo Paso a Paso:** A diferencia de las calculadoras tradicionales que solo dan el resultado final, esta herramienta genera la explicación del procedimiento, incluyendo la aplicación de la Regla de Sarrus y la expansión por cofactores (Laplace).
* **Prevención de Errores:** Bloqueo dinámico de dimensiones de matrices (ej. forzando matrices cuadradas para determinantes).

## 🛠️ Tecnologías Utilizadas

* **HTML5 & CSS3:** Interfaz limpia con CSS Grid, Flexbox y diseño 100% responsivo.
* **JavaScript (Vanilla):** Lógica matemática pura, algoritmos recursivos para cálculo de determinantes n-dimensionales y manipulación dinámica del DOM para renderizar las matrices y los pasos.
